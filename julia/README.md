### Set-up Env
- Env: WLS2 Ubuntu-20.04
- `wget https://julialang-s3.julialang.org/bin/linux/x64/1.7/julia-1.7.1-linux-x86_64.tar.gz`
- `tar zxvf julia-1.7.1-linux-x86_64.tar.gz`
- `mv julia-1.7.1 ~/`
- `export PATH="~/julia-1.7.1/bin:$PATH"`
- `source ~/.bashrc`
- `julia`
